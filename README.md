# m3c-proxy
Caching proxy server for a Triple Pattern Fragments server

## Quickstart

```bash
$ git clone https://github.com/ctsit/m3c-proxy
$ cd m3c-proxy
$ git checkout develop
$ npm install
$ npm run compile
$ npm start
```

## Deploy

After compiling, copy the whole `m3c-proxy` folder to the server and run:
```bash
$ npm start
```
